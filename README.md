A C# Windows Forms application that allows users to send bulk emails with a CV attachment using MimeKit and MailKit. Users can customize the email content, recipient list, and attachments.

Features

Send emails to multiple recipients from a CSV file.

Attach a CV or any document to the email.

Customizable email body (name, position, phone number, etc.).

Supports Gmail SMTP with App Password.

Logs email progress in the console or form.
